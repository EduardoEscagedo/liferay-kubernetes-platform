# Liferay on Kubernetes – Production Platform

This repository demonstrates a production-oriented deployment of Liferay DXP on Kubernetes.

## Objectives
- Deploy Liferay on Kubernetes
- Use Infrastructure as Code
- Add monitoring and CI/CD
- Follow production best practices

## Architecture
- Kubernetes cluster (1 control-plane, 1 worker)
- Liferay DXP on JBoss
- Ingress controller
- Persistent storage
- Monitoring stack

## Project Structure
k8s/        Kubernetes manifests
docs/       Architecture documentation
cicd/       CI/CD pipelines
scripts/    Helper scripts
